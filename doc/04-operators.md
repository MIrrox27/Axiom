    <h4>if</h4>
</div>

Когда мы хотим проверить какое-либо значение, мы можем использовать оператор 
```if```. Для работы с ним нам нужно сначала написать имя этого оператора (```if```),
потом идет проверяемое условие в круглых скобках, и блок действий в фигурных скобках. 
Пример:
```Axiom
if (x == true){
    print("x = ", x)
}

```
