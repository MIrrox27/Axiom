</div>


<div align="center">
    <h4>if</h4>
</div>

Когда мы хотим проверить какое-либо значение, мы можем использовать оператор 
```if```. Для работы с ним нам нужно сначала написать имя этого оператора (```if```),
потом идет проверяемое условие в круглых скобках, и блок действий в фигурных скобках. 

Пример:
```Axiom
if (x >= 1){
    print("x Больше 1 и равыен ", x) 
}

```


<div align="center">
    <h4>elif</h4>
</div>


Поле ```if``` можно написать неограниченное количество блоков ```elif```. Описывается он 
так же как и ```if```. 

Пример:

```Axiom
if (x > 1){
    print("x Больше 1 и равыен ", x) 
}

elif (x < 1){
    print("x меньше 1 и равен ", x)
}

elif (x == 1){
    print("x равен ", x)
}
```
